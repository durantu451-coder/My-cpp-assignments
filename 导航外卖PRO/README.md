# 导航外卖PRO

## 题目描述

这是一个5x5网格的导航外卖配送问题。程序需要从起点(0, 0)出发，找到一条到达终点(标记为9)的路径，并计算路径上的总时间。

## 网格说明
- `0`: 不可通过的障碍
- `1-8`: 可通过区域，数值表示通过该格子需要花费的时间
- `9`: 终点

## 解题思路

本程序采用**回溯法**进行路径搜索：

1. **数据结构**
   - 使用二维数组`a[N][N]`存储网格信息
   - 使用`visited[N][N]`记录已访问过的格子
   - 使用`vector<pair<int, int>> path`保存路径
   - 使用`t`记录总时间

2. **搜索策略**
   - 从起点(0, 0)开始，尝试向四个方向（右、下、上、左）移动
   - 使用递归进行深度优先搜索
   - 如果找到终点(9)，返回true并输出结果
   - 如果路径走不通，进行回溯（取消访问标记、减少时间、弹出路径）

3. **边界条件**
   - 坐标必须在0-4范围内
   - 不能访问值为0的格子或已访问过的格子

## 输入输出

**输入**: 5行5列的整数，每行5个数字

**输出**:
- 找到路径：输出总时间和路径坐标
- 未找到路径：输出"找不到"
