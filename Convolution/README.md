# 卷积实现程序

## 题目介绍

本程序实现了一个简单的二维卷积操作。给定一个固定的 3x3 卷积核和输入矩阵，程序将对输入矩阵进行卷积运算并输出结果。

### 输入格式
- 首先输入两个整数 x 和 y，表示输入矩阵的行数和列数
- 然后输入 x 行 y 列的整数矩阵

### 输出格式
- 输出卷积后的 x 行 y 列结果矩阵

### 卷积核
程序使用固定的卷积核：
```
[ 1, -1,  1]
[ 1, -1, -1]
[ 1, -1, -1]
```

## 解题思路

1. **输入处理**：首先读取输入矩阵的尺寸 x 和 y，然后读取矩阵元素

2. **边界扩展**：为了方便处理边界情况，创建一个比原矩阵大 2 的新矩阵，周围填充 0（即使用零填充）

3. **卷积计算**：
   - 遍历原矩阵的每个元素位置
   - 对于每个位置，使用 3x3 卷积核进行计算
   - 将对应位置的元素相乘后累加得到卷积结果

4. **结果输出**：按要求格式输出卷积后的矩阵

### 实现细节
- 使用 `vector<vector<int>>` 存储矩阵数据
- 通过双重循环实现卷积核的滑动操作
- 对于边界位置的处理，通过零填充避免了越界访问问题
